Maze Agent — Value Iteration

Draw a maze, place walls, then run the agent. The solver uses Value Iteration (Markov Decision Process) to compute the optimal policy — If no exit exists, it shows the best path the agent could find.

Algorithm: Value Iteration on a deterministic MDP. Rewards: +100 at goal, −0.1 per step, −5 for hitting a wall. Discount factor γ = 0.99. Converges when max Δ < 10⁻⁶.

No exit: If the agent loops or gets stuck, it traces the longest non-repeating path it could follow and reports the best value it achieved.
